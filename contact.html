Discord sunucumuza katılarak topluluğumuzun bir parçası olabilirsiniz. Ayrıca GitHub projelerimize katkıda bulunarak da aktif olarak yer alabilirsiniz.

Hangi teknolojiler üzerinde çalışıyorsunuz?

Ağırlıklı olarak web teknolojileri (HTML, CSS, JavaScript, React, Node.js), Python, veritabanları ve DevOps araçları üzerinde çalışıyoruz.

Yeni başlayan biri olarak katkıda bulunabilir miyim?

Kesinlikle! Yeni başlayanlar için özel projelerimiz ve mentorluk programımız var. Toplulukta öğrenme odaklı bir yaklaşım benimsiyor.

Projelerinize nasıl katkıda bulunabilirim?

GitHub projelerimizi inceleyerek issue'lara bakabilir, pull request gönderebilir veya yeni proje fikirlerinizi paylaşabilirsiniz.
